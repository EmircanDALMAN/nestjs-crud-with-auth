## Nestjs Crud Api With Auth

# yarn install

# To Run The Api
  ## yarn db:dev:up

# To Run The Tests
  ## db:test:up
  ## yarn test:e2e

# Used technologies
  ### Nestjs
  ### Prismajs -> For Crud and Database Operations
  ### Argon2 -> To Encrypt Passwords
  ### Passportjs / Passport-jwt -> Jwt Authentication
  ### Class validator -> For Model Validation
